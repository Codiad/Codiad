Codiad Plugins


Below are plugins tested and approved by the Codiad team. To install simply download and place the plugin's folder in the /plugins directory in Codiad. Then in the right-hand bar use the Plugins tool to enable the plugin. You can also easily write your own.

Name Description Author Download
ColorPicker Simple HEX and RGB color utility Fluidbyte |
Terminal Allows execution of terminal command through Codiad Fluidbyte |
DiffViewer Allows user to compare 2 files and to see all changes made in it daeks |
AutoUpdate Hooks into the Codiad updater to allow for automatic update daeks |
GitAdmin Allows for pulling repos into your current project daeks |
Codiad-Specific Dev Utils
PluginTemplate Boilerplate for developing Codiad Plugins Fluidyte |
PullTester Util to view and pull requests from Codiad core daeks |

Please note: While we test plugins before adding them to this list we cannot garuntee quality or provide any warranty. If you have any issues please address them directly with the plugin author.

Submit a Plugin

If you would like to submit a plugin, please email the GitHub repository and information about your plugin to dev[at]codiad.com.

Not sure how to get started? Check out the Codiad Plugin Template.
